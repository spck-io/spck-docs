# Spck Docs

Markdown sources and shared assets for [docs.spck.io](https://docs.spck.io).

Built and published from the main `createjsapp` repo via `gulp buildDocs` and `scripts/publish-docs.sh`. This repo is consumed as a git submodule at `createjsapp/docs/`.

## Layout

- `*.md` — doc pages, one per slug per locale (e.g. `cli-start.md`, `cli-start-de.md`)
- `assets/` — shared assets (gifs, images) hosted at `docs.spck.io/assets/...`
